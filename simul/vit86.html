威86

欢迎来到威86

这是一个能在浏览器上运行的虚拟机，支持运行多种远古/轻量级操作系统。你可以在左侧的菜单栏里选择一个操作系统来体验。

本网页基于V86制作而成。V86是一个能在网页上运行的x86虚拟机模拟器，1比1仿真模拟i386硬件配置，能运行一些32位操作系统。不需要安装任何软件，也不需要登录就能使用，大幅度减少资源占用，而且还不占内存，适合怀旧。

V86不是伪系统模拟，而是运行操作系统本身（系统执行一条CPU指令，V86逐条翻译、全程模拟运算；系统读写硬盘端口，V86操作内存里的镜像；系统输出图形到VGA显存，V86直接渲染到网页，所有硬件协议、中断、IO全是标准兼容。操作系统拿到的就是一台标准的x86硬件接口，根本就不知道自己其实是在浏览器上跑）。所以你在本网页体验到的所有系统，都是把真实镜像挂在上面的。

如果你也想要做这样的项目...

实现V86虚拟机并不困难，下面是一个简单的JavaScript代码，前提是你先要在html中引入v86的js。 
		
var emulator = window.emulator = new V86({
    wasm_path: "https://cdn.jsdelivr.net/npm/v86@0.5.419/build/v86.wasm",
    memory_size: 32 * 1024 * 1024, //内存大小根据情况调整，以字节为单位，下同。
    vga_memory_size: 2 * 1024 * 1024,
    screen_container: document.getElementById("screen"), //屏幕容器，类型要求是canvas。
    hda: { //img一般是hda、fda，iso一般是cdrom。
        url:'path/to/your/system', //映像源文件，注意跨域问题。
    },
    bios: {
        url: "https://cdn.jsdelivr.net/npm/v86-system@0.5.0/assets/seabios.bin",
    },
    vga_bios: {
        url: "https://cdn.jsdelivr.net/npm/v86-system@0.5.0/assets/vgabios.bin",
    },
    autostart: true //自动开机
});
